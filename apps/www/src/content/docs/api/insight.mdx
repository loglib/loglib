---
title: Insight
---

This endpoint provides similar data to your Insight dashboard. It can be used to display calculated data in your own admin dashboard or website.


## Get Insight

```bash
GET https://api.loglib.io
```

### Parameters

| Name | Type | Description |
|------|------|-------------|
startDate | string | The start date of the data to retrieve. Format: YYYY-MM-DDT00:00:00
endDate | string | The end date of the data to retrieve. Format: YYYY-MM-DDT00:00:00
timeZone | string | The timezone to use for the data. Format: Continent/City
apiKey | string | API key of the website you want to fetch. You can create one it in your [dashbaord](https://loglib.io/dashboard/api-keys).
